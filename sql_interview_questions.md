## 1. What is SQL?
- SQL stands for Structured Query Language
- Used to store, retrieve, update, and delete data
- Works with relational databases
- Allows communication between users and databases

---

## 2. What is a database?
- Organized collection of data
- Stored in tables
- Rows = records
- Columns = attributes
- Helps manage large data efficiently

---

## 3. Difference between SQL and MySQL?
- SQL = language
- MySQL = database system
- SQL gives instructions
- MySQL executes them

---

## 4. What is a primary key?
- Unique identifier of each record
- Cannot be NULL
- Cannot be duplicated
- Example: customer_id

---

## 5. What is a foreign key?
- Links two tables
- Refers to primary key in another table
- Maintains data consistency

---

## 6. What is a JOIN?
- Combines data from multiple tables
- Uses common column
- Types:
  - INNER JOIN
  - LEFT JOIN
  - RIGHT JOIN

---

## 7. What is GROUP BY?
- Groups similar values
- Used with aggregate functions
- Summarizes data

---

## 8. Aggregate functions
- SUM()
- AVG()
- MAX()
- MIN()
- COUNT()

---

## 9. What is normalization?
- Reduces duplication
- Organizes data
- Improves efficiency
- Ensures consistency

---

## 10. What is an index?
- Speeds up queries
- Like book index
- Too many indexes slow inserts

---

## 11. What is a view?
- Virtual table
- Created using SELECT
- Does not store data permanently
- Improves security and simplicity

---

## 12. Difference between WHERE and HAVING?
- WHERE filters rows
- HAVING filters grouped data

---

## 13. Why are joins important?
- Connect related tables
- Prevent duplication
- Useful in analysis and reporting
